# Banking Risk Analytics Dashboard

## Project Overview
This project focuses on analyzing banking customer data to help financial institutions reduce lending risks. The dashboard provides insights into customer income, deposits, engagement time, and investment patterns.

## Tools Used
- Power BI
- Python
- Data Cleaning
- Data Visualization

## Problem Statement
To understand how risk analytics can help banks minimize financial losses while lending to customers.

## Solution
A Power BI dashboard was developed to analyze customer profiles and determine loan approval possibilities based on financial indicators.

## Dataset Tables
- Banking Relationship
- Client-Banking
- Gender
- Investment Advisor
- Period

## Data Cleaning
- Created Engagement Timeframe
- Calculated Engagement Days
- Created Income Bands
- Processing Fee calculation

## Key Dashboard Insights
- Bank deposit analysis
- Customer engagement duration
- Income band segmentation
- Risk-based loan evaluation
